实时查看内存数据

在Xcode菜单栏中通过以下顺序可以查看对象的内存数据: 

Debug -> Debug Workfllow -> View Memory

对一个NSObject实例对象来说，通过该方法观察到前8个字节为非0值，后8个字节为全为0，第17位开始为非0值。 由此可以推断，前8个字节代表isa指针，也从侧面验证了一个NSObject实例对象指针指向内存的大小为16个字节。
